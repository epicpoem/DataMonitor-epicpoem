# FEATURE-02: 메인 메뉴 및 프로그램 흐름

---

## 기능 목적

사용자가 모니터링 도구를 실행하면 메인 메뉴가 표시되고, 원하는 조회 기능을 선택하거나 종료할 수 있다.

---

## 사용자 시나리오

1. 사용자가 프로그램을 실행한다.
2. 시스템이 메인 메뉴를 출력한다.
3. 사용자가 메뉴 번호를 입력한다.
4. 시스템이 선택한 기능을 실행한다.
5. 기능 실행 완료 후 메인 메뉴로 돌아간다.
6. 사용자가 종료(`[0]`)를 선택하면 프로그램이 종료된다.

---

## 메뉴 구성

| 번호 | 메뉴 | 설명 |
|------|------|------|
| `[1]` | 시료 전체 조회 | 등록된 모든 시료 목록 출력 |
| `[2]` | 주문 전체 조회 | 등록된 모든 주문 목록 출력 (선택 구현) |
| `[0]` | 종료 | 프로그램 종료 |

---

## 예외 처리 기준

- 메뉴에 없는 번호 또는 잘못된 입력: "잘못된 입력입니다. 다시 선택해주세요." 메시지를 출력하고 메뉴를 재표시한다.
