# FEATURE-01: 시료 전체 조회

---

## 기능 목적

DataPersistence에 저장된 모든 시료 데이터를 콘솔에서 한눈에 확인할 수 있도록 표 형식으로 출력한다.

---

## 사용자 시나리오

1. 사용자가 메인 메뉴에서 `[1] 시료 전체 조회`를 선택한다.
2. 시스템이 저장소에서 최신 시료 데이터를 읽는다.
3. 시스템이 전체 시료 목록을 표 형식으로 출력한다.
4. 표 하단에 총 건수를 표시한다.
5. 사용자가 확인 후 메인 메뉴로 돌아간다.

---

## 출력 항목

| 항목 | 설명 |
|------|------|
| ID | 시료 고유 식별자 (예: S-001) |
| 이름 | 시료 명칭 (예: 실리콘 웨이퍼-8인치) |
| 평균 생산시간 | 1개당 생산 소요 시간 (단위: min/ea) |
| 수율 | 정상 시료 비율 (예: 0.92) |
| 재고 | 현재 보유 수량 (단위: ea) |

---

## 예외 처리 기준

- 저장된 시료가 없는 경우: "등록된 시료가 없습니다." 메시지를 출력한다.
- 데이터 파일을 읽을 수 없는 경우: 오류 원인을 안내하는 메시지를 출력하고 메인 메뉴로 복귀한다.
